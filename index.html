O método de seleção

Um dos algoritmos mais simples

A ordenação por seleção (do inglês, selection sort) é um algoritmo de ordenação baseado em se passar sempre o menor valor do vetor para a primeira posição (ou o maior dependendo da ordem requerida), depois o de segundo menor valor para a segunda posição, e assim é feito sucessivamente com os (n-1) elementos restantes, até os últimos dois elementos.

Pricípios

  1. Seleciona o item com a menor chave;
  2. Seleciona o item com a menor chave;
  3. Troca este item com o item A [1];
  4. Repita a operação com os n − 1 itens restantes, depois com os n − 2
  5. Restantes, etc.

Observações

  • Vantagens:
    • É um dos métodos mais simples de ordenação existentes.
    • Para arquivos com registros muito grandes e chaves pequenas, este deve ser o método a ser utilizado.
    • Com chaves do tamanho de uma palavra, este método torna-se bastante interessante para arquivos pequenos.
  • Desvantagens:
    • O fato do arquivo já estar ordenado não ajuda em nada pois o custo continua quadrático.
    • O algoritmo não é estável, pois ele nem sempre deixa os registros com chaves iguais na mesma posição relativa.
